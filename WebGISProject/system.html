System Components

Real-time Weather and UV Index

Users can search for a location using the search bar, and upon selecting a location, a marker is placed on the map. Real-time weather and UV index information for that location is displayed via a popup. Additionally, users can choose to view either weather or UV index data using radio buttons.

Temperature Heatmap

The T-Heatmap button allows users to generate a heatmap that visualizes temperature variations across a selected area. Upon clicking, the system automatically creates the heatmap for the user to explore.

Data Table and CSV Download

The data table stores information about locations users have searched for or interacted with, including location name, latitude, longitude, temperature, and humidity. Users can download this data in CSV format for further analysis. The table is updated automatically when a user drags a marker to a new location.

Air Quality Index

The system displays real-time air quality information for a location, visually represented by a color-coded scale. A legend explains the classification of air quality, and users can view detailed data for various pollutants.

Geolocation

The system allows users to visualize their current location on the map and switch between different map layers. Users can input an address to get the corresponding coordinates or input coordinates to reverse geocode and display the address.

Facilities Search

Users can input their desired location coordinates and radius, and the system will create a buffer zone. It identifies and displays facilities like bus stops, hospitals, schools, and restaurants within the buffer zone, providing detailed information about these services.

Administrative Boundary and Road Network Visualization

The system integrates geographic data layers from GeoServer, including Nepal's district and administrative boundaries. It also visualizes the road network within Nepal, offering users the ability to explore various types of roads. Topographic information, such as Digital Elevation Models (DEM), and thematic maps at the provincial and district levels are also displayed.
