User Documentation

Login Page

New users can create an account by clicking "Create Account" and filling in their details. Afterward, log in using your credentials.

Main Page

Once logged in, the Main Page provides information about Geotasker and its features, along with details about the developer team.

Get Started

Click the Get Started button to proceed to the Main Dashboard.

Main Dashboard

The Main Dashboard contains various components and sections that can be accessed and interacted with.

Components and Functionalities

Each component offers distinct functionalities such as Real-time Weather, UV Index, Route Instructions, and more.

Section Buttons

Click the dedicated buttons for each section to access detailed information and functionalities.

Additional Information

If user wants to know about the GeoTasker objectives,Technology Stack, APIs then goto the bottom page of dashboard and explore.

Rating Option

At the end of the experience, users can rate Geotasker based on their experience using the app.
