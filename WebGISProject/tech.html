Technology Stack

Frontend

We use HTML, CSS, and JavaScript to build a responsive and interactive interface, allowing users to explore and interact with spatial data seamlessly.

Backend

The backend is powered by JavaScript (Node.js), handling server-side logic and API integrations efficiently. Additionally, for user login authentication, we use Django, a robust Python framework, ensuring secure and reliable user management.

APIs

We integrate external APIs for real-time data, including:

  • OpenWeatherMap API: For weather, air quality, and UV index data.
  • PositionStack API: For geolocation services to retrieve user location coordinates.
  • OpenStreetMap API: For facilities.
  • Open Service Route API: Forroute instruction.
